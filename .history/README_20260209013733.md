# Landing Page – Desafio Técnico Ellos Design

## Descricao do projeto
- Landing page premium/minimalista
- Desenvolvida em HTML, CSS e JavaScript puro
- Animacoes suaves com foco em performance
- Organizacao por blocos (BEM + `blocks/`)
- Inspirada em experiencias modernas de estudios digitais

## Como rodar o projeto

### Frontend (visualizacao)
1. Abra `index.html` diretamente no navegador.
2. Live Server pode ser usado apenas para visualizar o layout.
3. Live Server **nao executa PHP**.

### Full-Stack (backend funcional)
1. Execute o projeto via Apache + MySQL (XAMPP).
2. URL local utilizada neste projeto:

```text
http://localhost/teste-frontend-developer/
```

## Implementacao de performance
- `IntersectionObserver` para animacoes no scroll
- Cursor customizado
- `requestAnimationFrame` nas interacoes animadas
- Respeito a `prefers-reduced-motion`
- Sem bibliotecas externas pesadas

## Backend - PHP + MySQL
- O formulario envia os dados para `server/save_lead.php`
- O backend utiliza PHP com PDO
- Os dados sao gravados no MySQL
- O arquivo `server/schema.sql` cria o banco e a tabela
- Banco utilizado: `ellos_leads`
- A validacao pode ser conferida via phpMyAdmin ou SQL

## Passo a passo para validar o backend
1. Inicie Apache e MySQL no XAMPP.
2. Importe `server/schema.sql` no phpMyAdmin (ou MySQL CLI).
3. Acesse o site:

```text
http://localhost/teste-frontend-developer/
```

4. Envie o formulario na secao de contato.
5. No DevTools > Network, confirme a chamada para `server/save_lead.php` com resposta:

```json
{ "ok": true }
```

6. Confirme no banco:

```sql
SELECT * FROM leads ORDER BY created_at DESC;
```
